Structure des Sprites.

frame : Frame courante du sprite
frameNumber : Frame finale/maximal du sprite
spriteSheet : Chemin du spriteSheet
spriteSize : Taille en pixel du sprite
spriteLine : Indice de ligne du sprite dans la spriteSheet
