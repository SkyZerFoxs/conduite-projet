Description détaillée

Structure des Sprites.

Contient:
x : Coordonnées x du sprite
y : Coordonnées y du sprite
frame : Frame courante du sprite
spriteTypeId : Id du type de sprite à afficheer (sprite_type_liste_t)
monstre : Pointeur sur monstre_t du monstre relier au sprite ( NULL si aucun monstre )
pnj : Pointeur sur pnj_t du pnj relier au sprite ( NULL si aucun pnj )
