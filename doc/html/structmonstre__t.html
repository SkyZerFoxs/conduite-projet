Structure des monstres ( Data )

Contient:
char * nom, nom du monstre
int niveau, niveau du monstre
caract_t * caract, strcuture caract_t qui contient les statistique du monstre
int pos_y, position y du monstre dans la map
int pos_x, position x du monstre dans la map
int monstreSize, taille du monstre en nombre de sprite utilisé pour le former
