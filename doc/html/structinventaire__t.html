Description détaillée

Structure de l'inventaire du personnage ( Data & Affichage )

Contient:
int **inventaire, un pointeur de pointeur vers un tableau 2D d'entiers représentant l'inventaire
int nbPlaceInventaire, le nombre total de places dans l'inventaire
int invHeight, la hauteur de l'inventaire
int invWidth, la largeur de l'inventaire
int **equipement, un pointeur de pointeur vers un tableau 2D d'entiers représentant l'équipement
int nbPlaceEquipement, le nombre total de places dans l'équipement
int equipementHeight, la hauteur de l'inventaire d'équipement
int equipementWidth, la largeur de l'inventaire d'équipement
SDL_Texture * textInventaire, un pointeur vers une texture SDL représentant la texture de l'inteface d'inventaire
SDL_Texture * textItem, un pointeur vers une texture SDL représentant la texture des objets
SDL_Texture * textSelecteur, un pointeur vers une texture SDL représentant la texture du sélecteur
SDL_Texture * textItemInfo, un pointeur vers une texture SDL représentant la texture de l'inteface des informations d'un objet
int selecteurX, la position X du sélecteur
int selecteurY, la position Y du sélecteur
