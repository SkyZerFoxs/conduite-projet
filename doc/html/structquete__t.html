Structure des quetes ( Data )

Contient:
id, id de la quete
char * objectifQuete, texte description objectif de la quete
char * dialogueQuetePnj, texte du dialogue d'un pnj associé a la quete
quete_t * avant, pointeur sur la quete_t qui est avant
quete_t * apres, pointeur sur la quete_t qui est après
